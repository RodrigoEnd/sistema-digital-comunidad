========================================
SISTEMA DIGITAL DE GESTIÓN COMUNITARIA
========================================

INICIO RÁPIDO
=============

1. Instalar dependencias (primera vez):
   pip install -r requirements.txt

2. Ejecutar el sistema:
   python main.py

3. Seleccionar opción del menú:
   - 1: Censo de Habitantes
   - 2: Control de Pagos (requiere autenticación)
   - 3: Registro de Faenas (requiere autenticación)

Credenciales de demo:
   Usuario: admin
   Contraseña: admin123


ESTRUCTURA DE DIRECTORIOS
==========================

src/
├── auth/              # Autenticación y seguridad
├── core/              # Funciones core (BD, logger, validadores)
├── modules/           # Módulos principales
│   ├── censo/        # Censo de habitantes
│   ├── pagos/        # Control de pagos
│   ├── faenas/       # Registro de faenas
│   └── historial/    # Auditoría de cambios
├── tools/             # Backups y exportación
├── ui/                # Componentes de interfaz
└── config.py          # Configuración global


ALMACENAMIENTO
==============

Los datos se guardan en:
C:\Users\[usuario]\AppData\Local\SistemaComunidad\

Incluye:
- Base de datos de habitantes
- Registro de pagos
- Registro de faenas
- Historial de auditoría
- Backups automáticos
- Logs del sistema

Todos los datos están CIFRADOS con AES-256


CARACTERÍSTICAS DE SEGURIDAD
============================

✓ Autenticación de usuarios
✓ Cifrado AES-256 de datos
✓ Contraseñas hasheadas con bcrypt
✓ Auditoría de todos los cambios
✓ Historial de operaciones con timestamp
✓ Backups automáticos
✓ Datos guardados en ubicación protegida


AYUDA Y SOPORTE
===============

Para problemas:
- Revisar los logs en AppData\Local\SistemaComunidad\logs
- Usar reset_datos.py para reiniciar la base de datos
- Contactar al administrador del sistema
